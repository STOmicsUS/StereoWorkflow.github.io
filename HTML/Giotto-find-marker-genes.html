Find marker genes

Giotto provides three different methods to detect cluster specific marker genes: scran, gini and mast.
scran is generally recommended in spatial transcriptomics data. 


scran_markers = findScranMarkers_one_vs_all(
    giotto_object,
    expression_values = "normalized",
    cluster_column = "leiden_clus",
    pval = 0.01,
    logFC = 0.5,
    method = "scran",
    verbose = TRUE
)

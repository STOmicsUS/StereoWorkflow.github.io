StereoSeq Data Workflow

Created by Stomics US Data Team

This is a StereoSeq data analysis workflow. This demonstration will use a dataset SS200000135TL_D1 of a mouse brain.

Cell type deconvolution aims to estimate the proportion of different cell types present within each spatial spot, especially when each spot captures transcripts from multiple cells. This method leverages reference single-cell RNA-seq datasets to computationally infer cell type mixtures, enabling the spatial mapping of cellular composition across tissue sections.

Cell type annotation involves assigning a discrete cell type label to each spatial unit, such as a single cell or spot, based on its gene expression profile. This process typically uses reference datasets or computational methods to classify cells into known types, allowing for the identification of spatial patterns and functional states of cells within the tissue.

If you are using Stereo-Seq squrebin data, and your bin size is larger than a single cell, you should consider cell type deconvolution. If your are using cellbin data or your bin size is smaller than a single cell, you can use cell type annotation.
